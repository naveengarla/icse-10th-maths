# ICSE Class 10 Commercial Mathematics

## Complete Subtopics, Concepts and Formulae

3. **Shares and Dividends**

# Chapter 1 — GST

## 1. Important terminology

### Marked Price or List Price

The price written or printed on an article before giving a discount.

[
\text{Marked Price}=\text{M.P.}
]

### Discount

The amount deducted from the marked price.

[
\text{Discount}
===============

\frac{\text{Discount Rate}}{100}
\times
\text{Marked Price}
]

### Selling Price before GST

The price after subtracting the discount but before adding GST.

[
\boxed{
\text{Selling Price}
====================

\text{Marked Price}-\text{Discount}
}
]

Direct formula:

[
\boxed{
\text{Selling Price}
====================

\text{Marked Price}
\times
\frac{100-\text{Discount Rate}}{100}
}
]

### Taxable Value

For school-level GST questions:

[
\boxed{
\text{Taxable Value}=\text{Selling Price after Discount}
}
]

GST is ordinarily calculated on the discounted selling price, not on the original marked price.

---

## 2. Meaning of GST

GST is an indirect tax charged on the supply of goods and services.

For mathematical problems, the essential idea is:

[
\text{Amount before tax}
\longrightarrow
\text{Add GST}
\longrightarrow
\text{Final bill}
]

---

## 3. Direct tax and indirect tax

### Direct tax

A tax paid directly by the person on whom it is imposed.

Example: income tax.

### Indirect tax

A tax collected from the consumer through the seller.

GST is an indirect tax.

---

## 4. Intra-state transaction

An intra-state transaction occurs when the seller and buyer are in the same state or applicable Union Territory.

The total GST is divided into:

* CGST
* SGST

For a given total GST rate:

[
\boxed{
\text{CGST Rate}
================

# \text{SGST Rate}

\frac{\text{Total GST Rate}}{2}
}
]

For example, when total GST is (18%):

[
\text{CGST}=9%
]

[
\text{SGST}=9%
]

Therefore:

[
\boxed{
\text{Total GST}
================

\text{CGST}+\text{SGST}
}
]

---

## 5. Inter-state transaction

An inter-state transaction occurs when the seller and buyer are in different states or Union Territories.

IGST is charged.

[
\boxed{
\text{IGST Rate}=\text{Total GST Rate}
}
]

Example:

[
\text{Total GST}=18%
\implies
\text{IGST}=18%
]

Current school-textbook treatment commonly emphasizes CGST, SGST and IGST, although specific board questions may concentrate mainly on CGST and SGST calculations. ([Future Topper][2])

---

## 6. Computing GST

### Total GST amount

[
\boxed{
\text{GST Amount}
=================

\frac{\text{GST Rate}}{100}
\times
\text{Taxable Value}
}
]

### CGST amount

[
\boxed{
\text{CGST}
===========

\frac{\text{CGST Rate}}{100}
\times
\text{Taxable Value}
}
]

### SGST amount

[
\boxed{
\text{SGST}
===========

\frac{\text{SGST Rate}}{100}
\times
\text{Taxable Value}
}
]

### IGST amount

[
\boxed{
\text{IGST}
===========

\frac{\text{IGST Rate}}{100}
\times
\text{Taxable Value}
}
]

---

## 7. Final bill paid by consumer

### General formula

[
\boxed{
\text{Final Bill}
=================

\text{Selling Price}+\text{GST}
}
]

### Intra-state transaction

[
\boxed{
\text{Final Bill}
=================

\text{Selling Price}+\text{CGST}+\text{SGST}
}
]

### Inter-state transaction

[
\boxed{
\text{Final Bill}
=================

\text{Selling Price}+\text{IGST}
}
]

### Direct multiplier method

If the selling price before tax is known:

[
\boxed{
\text{Final Bill}
=================

\text{Selling Price}
\times
\frac{100+\text{GST Rate}}{100}
}
]

Example at (18%) GST:

[
\text{Final Bill}
=================

\text{Selling Price}\times\frac{118}{100}
]

---

## 8. Price inclusive of GST

Sometimes the given price already includes GST.

Let the inclusive price be (A).

[
\boxed{
\text{Price before GST}
=======================

A\times
\frac{100}{100+\text{GST Rate}}
}
]

### GST included in an inclusive price

[
\boxed{
\text{GST Amount}
=================

A-
\text{Price before GST}
}
]

Direct formula:

[
\boxed{
\text{GST Amount}
=================

A\times
\frac{\text{GST Rate}}
{100+\text{GST Rate}}
}
]

Example: ₹1,180 includes (18%) GST.

[
\text{Price before GST}
=======================

# 1180\times\frac{100}{118}

₹1000
]

[
\text{GST}=1180-1000=₹180
]

---

## 9. Finding the original price when the final bill is given

If the final amount after GST is known:

[
\boxed{
\text{Selling Price before GST}
===============================

\text{Final Amount}
\times
\frac{100}{100+\text{GST Rate}}
}
]

If the marked price must then be found and a discount was given:

[
\text{Selling Price}
====================

\text{Marked Price}
\times
\frac{100-d}{100}
]

Therefore:

[
\boxed{
\text{Marked Price}
===================

\text{Selling Price}
\times
\frac{100}{100-d}
}
]

where (d) is the discount percentage.

---

## 10. Successive discounts

When two discounts are given one after the other, they must not simply be added.

If discounts are (d_1%) and (d_2%):

[
\boxed{
\text{Final Price}
==================

\text{Marked Price}
\left(1-\frac{d_1}{100}\right)
\left(1-\frac{d_2}{100}\right)
}
]

Equivalent discount:

[
\boxed{
d_1+d_2-\frac{d_1d_2}{100}
}
]

For discounts of (10%) and (5%):

[
10+5-\frac{10\times5}{100}
==========================

14.5%
]

Not (15%).

---

## 11. Profit, loss and GST

GST is not normally included while calculating the trader’s profit or loss unless the question explicitly changes the meaning of cost.

### Profit

[
\boxed{
\text{Profit}
=============

## \text{Selling Price before GST}

\text{Cost Price before GST}
}
]

### Profit percentage

[
\boxed{
\text{Profit %}
===============

\frac{\text{Profit}}{\text{Cost Price}}
\times100
}
]

### Loss

[
\boxed{
\text{Loss}
===========

\text{Cost Price}-\text{Selling Price}
}
]

### Loss percentage

[
\boxed{
\text{Loss %}
=============

\frac{\text{Loss}}{\text{Cost Price}}
\times100
}
]

GST collected from the buyer is tax money, not the seller’s profit.

---

## 12. Input tax

Input tax is the GST paid by a dealer when purchasing goods.

[
\boxed{
\text{Input Tax}
================

\frac{\text{GST Rate}}{100}
\times
\text{Purchase Price}
}
]

---

## 13. Output tax

Output tax is the GST collected by a dealer when selling goods.

[
\boxed{
\text{Output Tax}
=================

\frac{\text{GST Rate}}{100}
\times
\text{Selling Price}
}
]

---

## 14. Input Tax Credit

The tax already paid while purchasing is allowed as a credit against the tax collected while selling.

[
\boxed{
\text{Input Tax Credit}=\text{Input Tax already paid}
}
]

---

## 15. Net GST payable

[
\boxed{
\text{Net GST Payable}
======================

\text{Output Tax}-\text{Input Tax Credit}
}
]

When the same GST rate applies to purchase and sale:

[
\text{Net GST}
==============

\frac{r}{100}
\times
\text{Selling Price}
--------------------

\frac{r}{100}
\times
\text{Purchase Price}
]

[
\boxed{
\text{Net GST}
==============

\frac{r}{100}
\left(
\text{Selling Price}-\text{Purchase Price}
\right)
}
]

Thus:

[
\boxed{
\text{Net GST}
==============

\frac{r}{100}
\times
\text{Value Added}
}
]

Input-tax-credit calculations are explicitly part of the GST chapter in common ICSE materials. ([BYJU'S][3])

---

## 16. Value added

[
\boxed{
\text{Value Added}
==================

\text{Selling Price}-\text{Purchase Price}
}
]

In simple textbook questions, this may coincide with profit when no other expenses are mentioned.

---

## 17. GST supply-chain problems

Typical chain:

[
\text{Manufacturer}
\rightarrow
\text{Wholesaler}
\rightarrow
\text{Retailer}
\rightarrow
\text{Consumer}
]

At each stage:

1. Calculate purchase price before GST.
2. Calculate input tax.
3. Calculate selling price before GST.
4. Calculate output tax.
5. Calculate net tax payable.
6. Calculate the buyer’s invoice amount.

The final consumer pays the complete GST and cannot claim input tax credit.

---

## 18. Invoice or bill problems

Questions may give:

* Item name
* Quantity
* Rate per item
* Discount condition
* GST rate
* Rounding instruction

For each article:

[
\text{Gross Value}
==================

\text{Quantity}\times\text{Rate}
]

[
\text{Discount}
===============

\frac{d}{100}\times\text{Gross Value}
]

[
\text{Taxable Value}
====================

\text{Gross Value}-\text{Discount}
]

[
\text{GST}
==========

\frac{r}{100}\times\text{Taxable Value}
]

[
\text{Article Total}
====================

\text{Taxable Value}+\text{GST}
]

Then:

[
\boxed{
\text{Grand Total}
==================

\text{Sum of all article totals}
}
]

Round only at the stage instructed in the question.

---

## 19. GST reduction or increase questions

If the GST rate falls from (r_1%) to (r_2%) and the base price remains unchanged:

[
\boxed{
\text{Reduction in Final Price}
===============================

\frac{r_1-r_2}{100}
\times
\text{Base Price}
}
]

Therefore:

[
\boxed{
\text{Base Price}
=================

\frac{\text{Reduction}\times100}{r_1-r_2}
}
]

---

## 20. Main GST question types

You should be able to solve:

* Marked price with discount and GST
* GST divided into CGST and SGST
* IGST calculations
* Final bill calculations
* Price inclusive of GST
* Reverse GST calculations
* Unknown GST rate
* Unknown marked price
* Unknown discount
* Profit or loss combined with GST
* Input tax credit
* Multi-stage dealer transactions
* Invoice verification
* GST-rate increase or decrease
* Successive discounts followed by tax

The syllabus description includes tax computation with discounts, list price, profit/loss, cost price and inverse cases. ([Future Topper][2])

# Chapter 2 — Banking: Recurring Deposit Accounts

The current ICSE Banking chapter primarily concerns **Recurring Deposit accounts**, especially calculation of interest and maturity value. The older savings-account ledger method found in some past papers or older books should not automatically be treated as part of the present RD-focused chapter. ([Shaalaa.com][4])

---

## 1. Recurring Deposit account

In an RD account, a fixed amount is deposited every month for a fixed number of months.

Let:

* (P) = monthly instalment
* (n) = number of months
* (r) = rate of interest per annum
* (I) = interest
* (M) or (MV) = maturity value

---

## 2. Total amount deposited

[
\boxed{
\text{Total Deposit}=P\times n
}
]

---

## 3. Why the instalments earn different interest

The first instalment remains deposited for the greatest number of months.

The last instalment remains deposited for the least number of months.

The equivalent month-count is:

[
n+(n-1)+(n-2)+\cdots+2+1
]

Using:

[
1+2+3+\cdots+n
==============

\frac{n(n+1)}{2}
]

the equivalent principal for one month becomes:

[
\boxed{
P\times\frac{n(n+1)}{2}
}
]

---

## 4. Interest on an RD account

[
\boxed{
I
=

P
\times
\frac{n(n+1)}{2}
\times
\frac{r}{100}
\times
\frac{1}{12}
}
]

Simplified:

[
\boxed{
I
=

\frac{P,n(n+1),r}{2400}
}
]

This is the fundamental RD formula used in the ICSE syllabus. ([Future Topper][2])

---

## 5. Maturity value

[
\boxed{
MV
==

\text{Total Deposit}+\text{Interest}
}
]

Therefore:

[
\boxed{
MV=Pn+I
}
]

Substituting the interest formula:

[
\boxed{
MV
==

Pn+
\frac{Pn(n+1)r}{2400}
}
]

---

## 6. Finding interest from maturity value

[
\boxed{
I=MV-Pn
}
]

This is often the first step when maturity value is given.

---

## 7. Finding monthly instalment

From:

[
I=
\frac{Pn(n+1)r}{2400}
]

[
\boxed{
P=
\frac{2400I}{n(n+1)r}
}
]

When maturity value is given, first use:

[
I=MV-Pn
]

Since (P) appears on both sides, substitute into:

[
MV
==

Pn+
\frac{Pn(n+1)r}{2400}
]

Factor out (P):

[
MV
==

P
\left[
n+\frac{n(n+1)r}{2400}
\right]
]

Hence:

[
\boxed{
P=
\frac{MV}
{n+\frac{n(n+1)r}{2400}}
}
]

---

## 8. Finding the rate of interest

[
I=
\frac{Pn(n+1)r}{2400}
]

Therefore:

[
\boxed{
r=
\frac{2400I}{Pn(n+1)}
}
]

---

## 9. Finding the number of months

From:

[
I=
\frac{Pn(n+1)r}{2400}
]

[
n(n+1)
======

\frac{2400I}{Pr}
]

Let:

[
K=\frac{2400I}{Pr}
]

Then find two consecutive positive integers whose product is (K).

For example:

[
n(n+1)=156
]

Since:

[
12\times13=156
]

[
\boxed{n=12}
]

When necessary, solve:

[
n^2+n-K=0
]

using factorisation or the quadratic formula.

---

## 10. Converting years into months

The RD formula uses (n) in months.

[
\boxed{
n=12\times\text{Number of Years}
}
]

Examples:

[
2\text{ years}=24\text{ months}
]

[
2\frac12\text{ years}=30\text{ months}
]

[
1\frac12\text{ years}=18\text{ months}
]

---

## 11. Effect of changing the monthly instalment

For fixed (n) and (r):

[
I\propto P
]

Therefore:

[
\boxed{
\frac{I_1}{I_2}
===============

\frac{P_1}{P_2}
}
]

If the monthly instalment increases by (20%), the interest also increases by (20%), provided the period and rate remain unchanged.

---

## 12. Effect of changing the rate

For fixed (P) and (n):

[
I\propto r
]

Therefore:

[
\boxed{
\frac{I_1}{I_2}
===============

\frac{r_1}{r_2}
}
]

---

## 13. Comparing two RD accounts

Calculate separately:

[
I_1=\frac{P_1n_1(n_1+1)r_1}{2400}
]

[
MV_1=P_1n_1+I_1
]

and:

[
I_2=\frac{P_2n_2(n_2+1)r_2}{2400}
]

[
MV_2=P_2n_2+I_2
]

Then compare interest, total deposit or maturity value as required.

---

## 14. RD word-problem clues

| Words in question  | Mathematical meaning |
| ------------------ | -------------------: |
| Monthly instalment |                  (P) |
| Period of deposit  |       (n), in months |
| Rate per annum     |                  (r) |
| Amount deposited   |                 (Pn) |
| Maturity value     |               (Pn+I) |
| Interest earned    |              (MV-Pn) |

---

## 15. Main Banking question types

You should be able to find:

* Total amount deposited
* Interest earned
* Maturity value
* Monthly instalment
* Rate of interest
* Number of months
* Period in years
* Change in interest when instalment changes
* Change in maturity value
* Comparison of two RD schemes
* Missing quantity using a quadratic equation

---

## 16. Frequent Banking mistakes

* Using years directly in place of months
* Forgetting (n+1)
* Using (1200) instead of (2400)
* Treating all instalments as if they earn interest for the full period
* Confusing total deposit with maturity value
* Forgetting:

[
I=MV-Pn
]

* Giving the number of months as the answer when the question asks for years

# Chapter 3 — Shares and Dividends

This chapter deals with face or nominal value, market value, premium, discount, dividend, investment, income and rate of return. Current syllabus summaries state that brokerage and fractional shares are excluded. ([Future Topper][2])

---

## 1. Share

A share represents one unit of ownership in a company.

A person who purchases shares is called a shareholder.

---

## 2. Nominal value or face value

The value printed on the share certificate.

[
\boxed{
\text{Nominal Value}
====================

# \text{Face Value}

# \text{N.V.}

\text{F.V.}
}
]

Dividend is calculated on nominal value, not market value.

---

## 3. Market value

The price at which a share is bought or sold in the market.

[
\boxed{
\text{Market Value}=\text{M.V.}
}
]

Investment is calculated using market value.

---

## 4. Share at par

A share is at par when:

[
\boxed{
MV=NV
}
]

Example:

A ₹100 share selling for ₹100 is at par.

---

## 5. Share at a premium

A share is above par when:

[
MV>NV
]

[
\boxed{
\text{Premium}=MV-NV
}
]

When the premium is (p%) of nominal value:

[
\boxed{
MV=NV\left(1+\frac{p}{100}\right)
}
]

or:

[
\boxed{
MV=NV\times\frac{100+p}{100}
}
]

---

## 6. Share at a discount

A share is below par when:

[
MV<NV
]

[
\boxed{
\text{Discount}=NV-MV
}
]

When the discount is (d%):

[
\boxed{
MV=NV\left(1-\frac{d}{100}\right)
}
]

or:

[
\boxed{
MV=NV\times\frac{100-d}{100}
}
]

---

## 7. Investment in shares

[
\boxed{
\text{Investment}
=================

\text{Number of Shares}
\times
\text{Market Value per Share}
}
]

Using symbols:

[
\boxed{
V=N\times MV
}
]

---

## 8. Number of shares purchased

[
\boxed{
N=
\frac{\text{Investment}}{\text{Market Value per Share}}
}
]

The number of shares must ordinarily be a whole number.

Fractional shares are not included in the current syllabus treatment. ([Future Topper][2])

---

## 9. Dividend

Dividend is the part of the company’s profit distributed to shareholders.

The dividend rate is calculated on the nominal value.

### Dividend per share

[
\boxed{
\text{Dividend per Share}
=========================

\frac{\text{Dividend Rate}}{100}
\times
NV
}
]

Example:

For a ₹100 share paying (12%) dividend:

[
\text{Dividend per share}
=========================

# \frac{12}{100}\times100

₹12
]

The shareholder receives ₹12 per share regardless of whether the share was purchased at ₹80, ₹100 or ₹150.

---

## 10. Annual income from shares

[
\boxed{
\text{Annual Income}
====================

\text{Number of Shares}
\times
\text{Dividend per Share}
}
]

Therefore:

[
\boxed{
\text{Annual Income}
====================

N
\times
\frac{r}{100}
\times
NV
}
]

where (r) is the dividend rate.

---

## 11. Total dividend paid by a company

If the company has issued (N) shares:

[
\boxed{
\text{Total Dividend}
=====================

N
\times
\frac{r}{100}
\times
NV
}
]

This uses the same structure as a shareholder’s income, but (N) represents all shares issued by the company.

---

## 12. Rate of return or yield

The return is based on the actual money invested.

[
\boxed{
\text{Rate of Return}
=====================

\frac{\text{Annual Income}}{\text{Investment}}
\times100
}
]

For one share:

[
\text{Income}
=============

\frac{r}{100}\times NV
]

[
\text{Investment}=MV
]

Therefore:

[
\boxed{
\text{Return %}
===============

\frac{r\times NV}{MV}
}
]

This is one of the most important formulas in the chapter.

---

## 13. Relationship between dividend rate and return

### Share at par

When:

[
MV=NV
]

then:

[
\boxed{
\text{Return Rate}=\text{Dividend Rate}
}
]

### Share above par

When:

[
MV>NV
]

then:

[
\boxed{
\text{Return Rate}<\text{Dividend Rate}
}
]

### Share below par

When:

[
MV<NV
]

then:

[
\boxed{
\text{Return Rate}>\text{Dividend Rate}
}
]

This relationship is frequently tested in reasoning-based MCQs.

## 14. Finding market value from return

From:

[
R=\frac{r\times NV}{MV}
]

[
\boxed{
MV=
\frac{r\times NV}{R}
}
]

where:

* (r) = dividend rate
* (R) = rate of return

---

## 15. Finding dividend rate from return

[
R=\frac{r\times NV}{MV}
]

Therefore:

[
\boxed{
r=
\frac{R\times MV}{NV}
}
]

---

## 16. Finding nominal value

[
R=\frac{r\times NV}{MV}
]

Therefore:

[
\boxed{
NV=
\frac{R\times MV}{r}
}
]

---

## 17. Finding investment from annual income and return

[
R=
\frac{\text{Income}}{\text{Investment}}\times100
]

Therefore:

[
\boxed{
\text{Investment}
=================

\frac{\text{Income}\times100}{R}
}
]

---

## 18. Finding income from investment and return

[
\boxed{
\text{Income}
=============

\frac{R}{100}
\times
\text{Investment}
}
]

---

## 19. Finding number of shares from annual income

First find dividend per share:

[
D=
\frac{r}{100}\times NV
]

Then:

[
\boxed{
N=
\frac{\text{Annual Income}}{\text{Dividend per Share}}
}
]

Therefore:

[
\boxed{
N=
\frac{100\times\text{Annual Income}}{r\times NV}
}
]

---

## 20. Finding investment required for a target income

First:

[
N=
\frac{\text{Target Income}}
{\text{Dividend per Share}}
]

Then:

[
\boxed{
\text{Investment}=N\times MV
}
]

Combined formula:

[
\boxed{
\text{Investment}
=================

\frac{\text{Target Income}\times MV\times100}
{r\times NV}
}
]

## 21. Selling shares

If (N) shares are sold at market value (MV_s):

[
\boxed{
\text{Sale Proceeds}
====================

N\times MV_s
}
]

If some shares remain:

[
\boxed{
\text{Remaining Shares}
=======================

\text{Original Shares}-\text{Shares Sold}
}
]

New annual income from the remaining shares:

[
\boxed{
\text{New Income}
=================

\text{Remaining Shares}
\times
\text{Dividend per Share}
}
]

---

## 22. Selling and reinvesting

Typical process:

### Step 1: Find existing shares

[
N_1=
\frac{\text{Original Investment}}{MV_1}
]

### Step 2: Find sale proceeds

[
S=N_1\times\text{Selling Price per Share}
]

### Step 3: Find new shares

[
N_2=
\frac{S}{MV_2}
]

### Step 4: Find new income

[
I_2=
N_2
\times
\frac{r_2}{100}
\times
NV_2
]

### Step 5: Compare incomes

[
\boxed{
\text{Change in Income}
=======================

\text{New Income}-\text{Old Income}
}
]

Positive result means an increase; negative result means a decrease.

---

## 23. Comparing two share investments

For each option, calculate return:

[
R_1=
\frac{r_1\times NV_1}{MV_1}
]

[
R_2=
\frac{r_2\times NV_2}{MV_2}
]

The share with the greater return percentage gives more annual income for the same investment.

Do not compare dividend rates alone.

## 24. Increase in income

If additional income required is (E):

[
\text{Additional Shares}
========================

\frac{E}{\text{Dividend per Share}}
]

Then:

[
\boxed{
\text{Additional Investment}
============================

\text{Additional Shares}\times MV
}
]

## 25. Percentage of shares sold

If (p%) of (N) shares are sold:

[
\boxed{
\text{Shares Sold}
==================

\frac{p}{100}\times N
}
]

[
\boxed{
\text{Shares Remaining}
=======================

N-\frac{p}{100}N
}
]

or:

[
\boxed{
\text{Shares Remaining}
=======================

N\times\frac{100-p}{100}
}
]

## 26. Profit or loss on selling shares

When brokerage is not involved:

[
\boxed{
\text{Profit}
=============

\text{Sale Proceeds}-\text{Original Investment}
}
]

[
\boxed{
\text{Loss}
===========

\text{Original Investment}-\text{Sale Proceeds}
}
]

[
\boxed{
\text{Profit %}
===============

\frac{\text{Profit}}{\text{Original Investment}}\times100
}
]

[
\boxed{
\text{Loss %}
=============

\frac{\text{Loss}}{\text{Original Investment}}\times100
}
]

Brokerage is excluded in the current listed syllabus. ([Future Topper][2])

# Essential distinction in Shares

This is the single most important conceptual table:

| Calculation         | Value used                                        |
| ------------------- | ------------------------------------------------- |
| Dividend per share  | Nominal value                                     |
| Annual income       | Nominal value                                     |
| Purchase price      | Market value                                      |
| Investment          | Market value                                      |
| Rate of return      | Income divided by actual investment               |
| Premium or discount | Difference between market value and nominal value |

Remember:

[
\boxed{\text{Dividend is calculated on N.V.}}
]

[
\boxed{\text{Investment is calculated on M.V.}}
]

# What appears outside these three chapters

A few items in the earlier collection need to be separated carefully:

* **Compound Interest** may be listed separately under Commercial Mathematics in some syllabus arrangements; it is not the same as the Recurring Deposit chapter. ([Shaalaa.com][4])

The best learning order is: GST basic bill calculations → reverse and ITC questions → RD direct formula questions → RD unknown-period questions → shares terminology → income and return → portfolio switching problems.
