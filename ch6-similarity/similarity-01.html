1. What Does "Similar" Actually Mean?

Start here — you already know this​

Have you ever:

  • Zoomed in on a Google Maps satellite image?
  • Seen a movie poster and the same image on a phone screen?
  • Printed a photo at two different sizes?

In all three cases — the image looks exactly the same, just at a different size. The shape didn't change. The proportions didn't change. Only the size changed.

That is similarity.

The one-sentence definition

Two shapes are similar if they have the same shape but not necessarily the same size.

What "same shape" actually means​

"Same shape" sounds obvious — but in mathematics it means something precise:

Every angle in one shape is equal to the corresponding angle in the other shape.

That's it. If all angles match — the shapes are similar.

Why angles and not sides? Because angles define the shape. Sides define the size. When you zoom a photo, you change the size (sides get bigger), but you never change the shape (angles stay the same).

  • The angles at each corner are identical across all three triangles
  • The side lengths are different — but they all grew by the same factor
  • The shape — how "pointy" or "flat" the triangle looks — is identical

This is what it means to be similar.

The two things that define similarity​

  • 📐 Equal Angles
  • 📏 Proportional Sides

In similar triangles, every angle in one triangle equals the corresponding angle in the other.

If △ABC∼△PQR\triangle ABC \sim \triangle PQR△ABC∼△PQR, then: ∠A=∠P,∠B=∠Q,∠C=∠R\angle A = \angle P, \quad \angle B = \angle Q, \quad \angle C = \angle R∠A=∠P,∠B=∠Q,∠C=∠R

Corresponding means they are in the same position — the "top" angle of one matches the "top" angle of the other.

This is not a coincidence. If the angles are the same, the triangles must be similar — the shape is completely determined by the angles.

In similar triangles, corresponding sides are in the same ratio.

If △ABC∼△PQR\triangle ABC \sim \triangle PQR△ABC∼△PQR, then: ABPQ=BCQR=CARP=k\dfrac{AB}{PQ} = \dfrac{BC}{QR} = \dfrac{CA}{RP} = kPQAB​=QRBC​=RPCA​=k

where kkk is called the scale factor.

This ratio kkk is the same for all three pairs of corresponding sides. That's the key — not that the sides are equal, but that they are all in the same ratio.

What changes and what stays the same​

PropertyStays the same?Why?
All angles✅ Yes — always equalAngles define the shape
Shape (how it looks)✅ Yes — identicalBecause angles are equal
Side lengths❌ No — they scaleSides define the size
Ratio of sides✅ Yes — all equal to kThat's what proportional means
Area❌ No — it changesArea depends on size

The notation​

When we write △ABC∼△PQR\triangle ABC \sim \triangle PQR△ABC∼△PQR, the order of letters matters.

It tells you exactly which vertex matches which:

  • A matches P
  • B matches Q
  • C matches R

So side ABABAB corresponds to side PQPQPQ, and so on.

Common mistake

△ABC∼△PQR\triangle ABC \sim \triangle PQR△ABC∼△PQR is NOT the same as △ABC∼△QPR\triangle ABC \sim \triangle QPR△ABC∼△QPR.

The order of letters tells you which angles are equal. Always check the order carefully.

Two triangles: one has angles 40°, 60°, 80°. The other has angles 60°, 80°, 40°.

Are they similar?

Yes — similar!

Both have the same three angles: 40°, 60°, 80°. The order they're listed doesn't matter — the shapes are identical.

The correct notation would be to match corresponding angles:
If ∠A=40°,∠B=60°,∠C=80°\angle A = 40°, \angle B = 60°, \angle C = 80°∠A=40°,∠B=60°,∠C=80° and ∠P=40°,∠Q=80°,∠R=60°\angle P = 40°, \angle Q = 80°, \angle R = 60°∠P=40°,∠Q=80°,∠R=60°, then △ABC∼△PRQ\triangle ABC \sim \triangle PRQ△ABC∼△PRQ.

Two triangles: AB=3AB = 3AB=3, BC=4BC = 4BC=4, CA=5CA = 5CA=5 and PQ=6PQ = 6PQ=6, QR=8QR = 8QR=8, RP=10RP = 10RP=10.

Are they similar?

Yes — similar with k = 2!

Check the ratios: PQAB=63=2\dfrac{PQ}{AB} = \dfrac{6}{3} = 2ABPQ​=36​=2, QRBC=84=2\dfrac{QR}{BC} = \dfrac{8}{4} = 2BCQR​=48​=2, RPCA=105=2\dfrac{RP}{CA} = \dfrac{10}{5} = 2CARP​=510​=2

All three ratios are equal to 2. So △ABC∼△PQR\triangle ABC \sim \triangle PQR△ABC∼△PQR with scale factor k=2k = 2k=2.

Also note: 3-4-5 and 6-8-10 are both Pythagorean triples — both are right-angled triangles.

Summary​

What to remember
  1. Similar = same shape, different (or same) size
  2. Angles are equal in similar figures
  3. Sides are proportional (in the same ratio k) — NOT necessarily equal
  4. The order of letters in △ABC∼△PQR\triangle ABC \sim \triangle PQR△ABC∼△PQR tells you which vertex matches which
