📐 Trigonometry Identities — Deep Learning

1. The Story (Finland Phenomenon)​

🏗️ Riya's Architecture Project

Riya is designing a ramp for a wheelchair. She knows the ramp must rise 1 metre over a horizontal run of 4 metres. What angle does it make? Is it safe? Her calculation uses only one idea — the ratio of two sides of a right triangle. That idea is trigonometry.



The identities in this chapter are the relationships between those ratios that are always true, no matter what the triangle looks like.

2. The Six Ratios — CPA Approach​

  • 🧱 Concrete
  • 🖼️ Pictorial
  • 🔢 Abstract

Imagine holding a right-angled set-square. Label one acute angle A.

  • The side across from A = Opposite (p)
  • The side next to A (not the longest) = Adjacent (b)
  • The longest side = Hypotenuse (h)

These three sides give six possible ratios. Each ratio has a name.

RatioFormulaReciprocal
sin Ap / hcosec A = h / p
cos Ab / hsec A = h / b
tan Ap / bcot A = b / p

Also: tan A = sin A / cos A and cot A = cos A / sin A

3. The Three Pythagorean Identities — CPA​

  • 🧱 Concrete
  • 🖼️ Pictorial
  • 🔢 Abstract

Pythagoras says: p² + b² = h² (always, for any right triangle).

Now just divide both sides by h², by b², or by p² — you get three different identities. That's literally all the "proof" is.

Divide p² + b² = h² by h²:

sin²A + cos²A = 1

Divide p² + b² = h² by b²:

1 + tan²A = sec²A

Divide p² + b² = h² by p²:

1 + cot²A = cosec²A

4. All Derived Forms (memorise these)​

  • From Identity 1
  • From Identity 2
  • From Identity 3

From sin²A + cos²A = 1:

sin²A = 1 − cos²A
cos²A = 1 − sin²A

From 1 + tan²A = sec²A:

sec²A − tan²A = 1
tan²A = sec²A − 1

From 1 + cot²A = cosec²A:

cosec²A − cot²A = 1
cot²A = cosec²A − 1

Pattern to remember: Each identity has the form 1 + (smaller)² = (bigger)². sin/cos → 1; tan/sec → sec² wins; cot/cosec → cosec² wins.

6. Formulas in 4 Languages​

Identity 1 — sin²A + cos²A = 1​

  • 🗣️ Words
  • 👁️ Visual
  • 🔣 Math
  • ✋ Try It

The square of sin and the square of cos always add up to exactly 1, for any angle A between 0° and 90°. This comes directly from Pythagoras — divide both sides of p²+b²=h² by h².

See the Pictorial tab in Section 3 — the triangle with unit hypotenuse makes this obvious. (sin A)² + (cos A)² = (p/h)² + (b/h)² = (p²+b²)/h² = h²/h² = 1.

sin²A + cos²A = 1
∴ sinA = √(1 − cos²A)
∴ cosA = √(1 − sin²A)

If sin A = 3/5, find cos A and tan A.

  • cos²A = 1 − sin²A = 1 − 9/25 = 16/25
  • cos A = 4/5
  • tan A = sin A / cos A = (3/5)÷(4/5) = 3/4

7. Worked Examples​

  • 📗 Prove Identity
  • 📙 Simplify
  • 📘 Given Ratio
  • 📕 Eliminate θ

Q1 — Prove: (1 − sin²A) / cos²A = 1

  1. LHS = (1 − sin²A) / cos²A
  2. Use sin²A + cos²A = 1 → 1 − sin²A = cos²A
  3. LHS = cos²A / cos²A = 1 = RHS ✓
✓ Proved

Q2 — Prove: sin⁴A + cos⁴A + 2sin²A cos²A = 1

  1. LHS = (sin²A + cos²A)² = 1² = 1 = RHS ✓ (Recognise the perfect square pattern: a²+b²+2ab = (a+b)²)
✓ Proved

Q3 — Simplify: (sec²A − 1) / sec²A

  1. sec²A − 1 = tan²A (from identity 2)
  2. = tan²A / sec²A
  3. = (sin²A/cos²A) / (1/cos²A) = sin²A
✓ Answer: sin²A

Q4 — Simplify: cosec A (1 − cos²A)

  1. 1 − cos²A = sin²A
  2. = cosec A × sin²A = (1/sin A) × sin²A = sin A
✓ Answer: sin A

Q5 — If tan A = 4/3, find sin A and cos A (A is acute)

  1. tan A = p/b = 4/3, so p = 4k, b = 3k
  2. h = √(p²+b²) = √(16k²+9k²) = 5k
  3. sin A = p/h = 4/5, cos A = b/h = 3/5
✓ sin A = 4/5 | cos A = 3/5

Q6 — If 3 cot A = 4, find (3 sin A − 4 cos A) / (3 sin A + 4 cos A)

  1. cot A = 4/3, so tan A = 3/4 → p=3, b=4, h=5
  2. sin A = 3/5, cos A = 4/5
  3. Numerator = 3(3/5) − 4(4/5) = 9/5 − 16/5 = −7/5
  4. Denominator = 3(3/5) + 4(4/5) = 9/5 + 16/5 = 25/5 = 5
  5. Answer = (−7/5) / 5 = −7/25
✓ Answer: −7/25

Q7 — If x = a sinθ and y = b cosθ, prove x²/a² + y²/b² = 1

  1. x/a = sinθ → x²/a² = sin²θ
  2. y/b = cosθ → y²/b² = cos²θ
  3. x²/a² + y²/b² = sin²θ + cos²θ = 1 ✓
✓ Proved (standard elimination pattern)
