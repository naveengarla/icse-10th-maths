# 📐 PYQ — Trigonometry Identities (2000–2025)

:::tip Strategy
**For Prove questions:** Work on ONE side only. If stuck — convert everything to sin and cos.
**For Evaluate questions:** Replace complementary angles (cos70° = sin20°, etc.) first, then simplify.
:::

## Year 2000

<PYQCard year="2000" marks="4" qnum="1"
  solution={<>
    <strong>Multiply inside √ by (1−cosA)/(1−cosA):</strong><br/>
    LHS = √[(1−cosA)² / (1−cos²A)] = √[(1−cosA)² / sin²A]<br/>
    = (1−cosA) / sinA<br/><br/>
    <strong>Show RHS equals same:</strong><br/>
    sinA/(1+cosA) × (1−cosA)/(1−cosA) = sinA(1−cosA)/(1−cos²A) = sinA(1−cosA)/sin²A = (1−cosA)/sinA ✓
  </>}
>
  Prove: √[(1−cosA)/(1+cosA)] = sinA/(1+cosA)
</PYQCard>

<PYQCard year="2000" marks="4" qnum="2"
  solution={<>
    <strong>Use complementary angle: cos(90°−A) = sinA, cosec(90°−A) = secA</strong><br/>
    LHS = 1/(1+sinA) + 1/(1−sinA)<br/>
    = (1−sinA + 1+sinA) / [(1+sinA)(1−sinA)]<br/>
    = 2 / (1−sin²A)<br/>
    = 2 / cos²A = 2sec²A = 2cosec²(90°−A) = RHS ✓
  </>}
>
  Prove: 1/(1+cos(90°−A)) + 1/(1−cos(90°−A)) = 2cosec²(90°−A)
</PYQCard>

## Year 2002

<PYQCard year="2002" marks="4" qnum="1"
  solution={<>
    LHS = [(sinA−cosA) + (sinA+cosA)] / [(sinA+cosA)(sinA−cosA)]<br/>
    = 2sinA / (sin²A − cos²A)<br/>
    = 2sinA / (1 − cos²A − cos²A)<br/>
    = 2sinA / (1 − 2cos²A) = RHS ✓
  </>}
>
  Prove: 1/(sinA+cosA) + 1/(sinA−cosA) = 2sinA/(1−2cos²A)
</PYQCard>

<PYQCard year="2002" marks="4" qnum="2"

    = (1−cosA + 1+cosA) / (1−cos²A)<br/>
    = 2 / sin²A = 2cosec²A = 2sec²(90°−A) = RHS ✓
  </>}
>
  Prove: 1/(1+sin(90°−A)) + 1/(1−sin(90°−A)) = 2sec²(90°−A)
</PYQCard>

## Year 2003

<PYQCard year="2003" marks="4" qnum="1"
  solution={<>
    <strong>Convert tan and cot to sin/cos:</strong><br/>
    = cosA/(1 − sinA/cosA) + sinA/(1 − cosA/sinA)<br/>
    = cos²A/(cosA−sinA) + sin²A/(sinA−cosA)<br/>
    = cos²A/(cosA−sinA) − sin²A/(cosA−sinA)<br/>
    = (cos²A − sin²A)/(cosA−sinA)<br/>
    = (cosA+sinA)(cosA−sinA)/(cosA−sinA)<br/>
    = cosA + sinA = RHS ✓
  </>}
>
  Prove: cosA/(1−tanA) + sinA/(1−cotA) = sinA + cosA
</PYQCard>

## Year 2004

<PYQCard year="2004" marks="4" qnum="1"
  solution={<>
    <strong>Convert to sin/cos:</strong><br/>
    LHS = (cosA/sinA + sinB/cosB) / (cosB/sinB + sinA/cosA)<br/>
    = [(cosAcosB + sinAsinB)/(sinAcosB)] / [(cosAcosB + sinAsinB)/(sinBcosA)]<br/>
    = (sinBcosA) / (sinAcosB)<br/>
    = (cosA/sinA) × (sinB/cosB)<br/>
    = cotA × tanB = RHS ✓
  </>}
>
  Prove: (cotA + tanB)/(cotB + tanA) = cotA tanB
</PYQCard>

<PYQCard year="2004" marks="3" qnum="2"
  solution={<>
    = 14 × sin30° + 6 × cos60° − 5 × tan45°<br/>
    = 14 × (1/2) + 6 × (1/2) − 5 × 1<br/>
    = 7 + 3 − 5<br/>
    = <strong>5</strong>
  </>}
>
  Evaluate: 14sin30° + 6cos60° − 5tan45°
</PYQCard>

## Year 2008

<PYQCard year="2008" marks="4" qnum="1"
  solution={<>

    = (1−cos²A) / [cosA(1−cosA)]<br/>

    = (1+cosA) / cosA<br/>
    = 1/cosA + cosA/cosA<br/>
    = secA + 1 = RHS ✓
  </>}
>

</PYQCard>

## Year 2018

<PYQCard year="2018" marks="4" qnum="1"
  solution={<>
    <strong>Square both sides (both positive for 0&lt;θ&lt;90°):</strong><br/>
    RHS² = (tanθ+cotθ)² = (sinθ/cosθ + cosθ/sinθ)²<br/>
    = [(sin²θ+cos²θ)/(sinθcosθ)]² = 1/(sin²θcos²θ)<br/><br/>
    LHS² = sec²θ + csc²θ = 1/cos²θ + 1/sin²θ<br/>
    = (sin²θ+cos²θ)/(sin²θcos²θ) = 1/(sin²θcos²θ)<br/><br/>
    LHS² = RHS² and both sides positive ∴ LHS = RHS ✓
  </>}
>
  Prove: √(sec²θ + csc²θ) = tanθ + cotθ
</PYQCard>

<PYQCard year="2018" marks="4" qnum="2"
  solution={<>
    <strong>Convert to sin/cos:</strong><br/>
    = [(sinθ + cosθ − 1)/sinθ] × [(cosθ + sinθ + 1)/cosθ]<br/>
    = (sinθ+cosθ−1)(sinθ+cosθ+1) / (sinθcosθ)<br/>
    = [(sinθ+cosθ)² − 1] / (sinθcosθ)<br/>
    = [sin²θ + 2sinθcosθ + cos²θ − 1] / (sinθcosθ)<br/>
    = [1 + 2sinθcosθ − 1] / (sinθcosθ)<br/>
    = 2sinθcosθ / sinθcosθ = 2 = RHS ✓
  </>}
>
  Prove: (1 + cotθ − cscθ)(1 + tanθ + secθ) = 2
</PYQCard>

## Year 2019

</PYQCard>

  </>}

</PYQCard>
