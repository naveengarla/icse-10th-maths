# 📐 PYQ — Trigonometric Identities (2000–2025)

:::tip Strategy
**Prove:** Work ONE side only. Stuck? Convert everything to $\sin A$ and $\cos A$.

**Evaluate:** Replace complementary pairs first — $\cos 70° = \sin 20°$, $\tan 72° = \cot 18°$.
:::

---

## Year 2000

<PYQCard year="2000" marks="4" qnum="1">

Prove: $\sqrt{\dfrac{1-\cos A}{1+\cos A}} = \dfrac{\sin A}{1+\cos A}$

<details>
<summary>Show Solution</summary>

Multiply inside $\sqrt{\phantom{x}}$ by $\dfrac{1-\cos A}{1-\cos A}$:

$$\text{LHS} = \sqrt{\dfrac{(1-\cos A)^2}{\sin^2 A}} = \dfrac{1-\cos A}{\sin A}$$

RHS: $\dfrac{\sin A}{1+\cos A} = \dfrac{\sin A(1-\cos A)}{\sin^2 A} = \dfrac{1-\cos A}{\sin A} \checkmark$

</details>

</PYQCard>

<PYQCard year="2000" marks="4" qnum="2">

Prove: $\dfrac{1}{1+\cos(90°-A)} + \dfrac{1}{1-\cos(90°-A)} = 2\operatorname{cosec}^2(90°-A)$

<details>
<summary>Show Solution</summary>

Use $\cos(90°-A)=\sin A$ and $\operatorname{cosec}(90°-A)=\sec A$:

$$\text{LHS} = \dfrac{1}{1+\sin A}+\dfrac{1}{1-\sin A} = \dfrac{2}{1-\sin^2 A} = \dfrac{2}{\cos^2 A} = 2\sec^2 A = 2\operatorname{cosec}^2(90°-A) \checkmark$$

</details>

</PYQCard>

## Year 2001

<PYQCard year="2001" marks="4" qnum="1">

Prove: $\sqrt{\dfrac{1-\sin A}{1+\sin A}} = \dfrac{\cos A}{1+\sin A}$

<details>
<summary>Show Solution</summary>

$$\text{LHS} = \sqrt{\dfrac{(1-\sin A)^2}{\cos^2 A}} = \dfrac{1-\sin A}{\cos A}$$

RHS $= \dfrac{\cos A(1-\sin A)}{\cos^2 A} = \dfrac{1-\sin A}{\cos A} = \text{LHS} \checkmark$

</details>

</PYQCard>

<PYQCard year="2001" marks="4" qnum="2">

If $2\sin A - 1 = 0$, show that $\sin 3A = 3\sin A - 4\sin^3 A$.

<details>
<summary>Show Solution</summary>

$2\sin A=1 \Rightarrow \sin A=\dfrac{1}{2} \Rightarrow A=30°$

**LHS:** $\sin 90° = 1$

**RHS:** $3\cdot\dfrac{1}{2}-4\cdot\dfrac{1}{8} = \dfrac{3}{2}-\dfrac{1}{2} = 1 \checkmark$

</details>

</PYQCard>

$$\text{LHS} = \dfrac{2\sin A}{\sin^2 A-\cos^2 A} = \dfrac{2\sin A}{1-\cos^2 A-\cos^2 A} = \dfrac{2\sin A}{1-2\cos^2 A} = \text{RHS} \checkmark$$

</details>

</PYQCard>

<PYQCard year="2002" marks="4" qnum="2">

Use $\sin(90°-A)=\cos A$:

$$\text{LHS} = \dfrac{1}{1+\cos A}+\dfrac{1}{1-\cos A} = \dfrac{2}{\sin^2 A} = 2\operatorname{cosec}^2 A = 2\sec^2(90°-A) \checkmark$$

</details>

</PYQCard>

## Year 2003

<PYQCard year="2003" marks="4" qnum="1">

Prove: $\dfrac{\cos A}{1-\tan A}+\dfrac{\sin A}{1-\cot A} = \sin A+\cos A$

<details>
<summary>Show Solution</summary>

$$= \dfrac{\cos^2 A}{\cos A-\sin A}+\dfrac{\sin^2 A}{\sin A-\cos A} = \dfrac{\cos^2 A-\sin^2 A}{\cos A-\sin A} = \cos A+\sin A = \text{RHS} \checkmark$$

</details>

</PYQCard>

## Year 2004

<PYQCard year="2004" marks="4" qnum="1">

Prove: $\dfrac{\cot A+\tan B}{\cot B+\tan A} = \cot A\tan B$

<details>
<summary>Show Solution</summary>

$$\text{LHS} = \dfrac{\frac{\cos A\cos B+\sin A\sin B}{\sin A\cos B}}{\frac{\cos A\cos B+\sin A\sin B}{\sin B\cos A}} = \dfrac{\sin B\cos A}{\sin A\cos B} = \cot A\tan B = \text{RHS} \checkmark$$

</details>

</PYQCard>

<PYQCard year="2004" marks="3" qnum="2">

<summary>Show Solution</summary>

$$= 14\times\dfrac{1}{2}+6\times\dfrac{1}{2}-5\times 1 = 7+3-5 = \boxed{5}$$

</details>

</PYQCard>

</details>

Prove: $\dfrac{\sin\theta-2\sin^3\theta}{2\cos^3\theta-\cos\theta} = \tan\theta$

<details>
<summary>Show Solution</summary>

$$\text{LHS} = \dfrac{\sin\theta(1-2\sin^2\theta)}{\cos\theta(2\cos^2\theta-1)}$$

Since $2\cos^2\theta-1 = 2(1-\sin^2\theta)-1 = 1-2\sin^2\theta$:

$$= \dfrac{\sin\theta}{\cos\theta} = \tan\theta = \text{RHS} \checkmark$$

</details>

</PYQCard>

## Year 2018

<PYQCard year="2018" marks="4" qnum="1">

Prove: $\sqrt{\sec^2\theta+\operatorname{cosec}^2\theta} = \tan\theta+\cot\theta$

<details>
<summary>Show Solution</summary>

$$\text{RHS}^2 = \left(\dfrac{1}{\sin\theta\cos\theta}\right)^2 = \dfrac{1}{\sin^2\theta\cos^2\theta}$$

$$\text{LHS}^2 = \dfrac{1}{\cos^2\theta}+\dfrac{1}{\sin^2\theta} = \dfrac{1}{\sin^2\theta\cos^2\theta}$$

Both sides positive ∴ $\text{LHS} = \text{RHS} \checkmark$

</details>

</PYQCard>

<PYQCard year="2018" marks="4" qnum="2">

$$= \dfrac{(\sin\theta+\cos\theta)^2-1}{\sin\theta\cos\theta} = \dfrac{2\sin\theta\cos\theta}{\sin\theta\cos\theta} = 2 = \text{RHS} \checkmark$$

</details>

</PYQCard>
