# 📐 PYQ — Trigonometric Identities (2000–2025)

:::tip Strategy
**Prove:** Work ONE side only. Stuck? Convert everything to $\sin A$ and $\cos A$.  
**Evaluate:** Replace complementary pairs first — $\cos 70° = \sin 20°$, $\tan 72° = \cot 18°$, etc.
:::

---

## Year 2000

<PYQCard year="2000" marks="4" qnum="1">

Prove: $\sqrt{\dfrac{1-\cos A}{1+\cos A}} = \dfrac{\sin A}{1+\cos A}$

<details><summary>Show Solution</summary><div>

**Multiply inside $\sqrt{\phantom{x}}$ by $\dfrac{1-\cos A}{1-\cos A}$:**

$$\text{LHS} = \sqrt{\dfrac{(1-\cos A)^2}{1-\cos^2 A}} = \sqrt{\dfrac{(1-\cos A)^2}{\sin^2 A}} = \dfrac{1-\cos A}{\sin A}$$

**Show RHS equals the same:**

$$\dfrac{\sin A}{1+\cos A} = \dfrac{\sin A(1-\cos A)}{1-\cos^2 A} = \dfrac{\sin A(1-\cos A)}{\sin^2 A} = \dfrac{1-\cos A}{\sin A} \checkmark$$

<div className="ans">✓ LHS = RHS = $\dfrac{1-\cos A}{\sin A}$</div>
</div></details>

</PYQCard>

<PYQCard year="2000" marks="4" qnum="2">

Prove: $\dfrac{1}{1+\cos(90°-A)} + \dfrac{1}{1-\cos(90°-A)} = 2\operatorname{cosec}^2(90°-A)$

<details><summary>Show Solution</summary><div>

Use: $\cos(90°-A) = \sin A$ and $\operatorname{cosec}(90°-A) = \sec A$

$$\text{LHS} = \dfrac{1}{1+\sin A} + \dfrac{1}{1-\sin A} = \dfrac{(1-\sin A)+(1+\sin A)}{(1+\sin A)(1-\sin A)} = \dfrac{2}{1-\sin^2 A} = \dfrac{2}{\cos^2 A} = 2\sec^2 A$$

$= 2\operatorname{cosec}^2(90°-A) = \text{RHS}$ $\checkmark$

</div></details>

</PYQCard>

## Year 2001

<PYQCard year="2001" marks="4" qnum="1">

Prove: $\sqrt{\dfrac{1-\sin A}{1+\sin A}} = \dfrac{\cos A}{1+\sin A}$

<details><summary>Show Solution</summary><div>

$$\text{LHS} = \sqrt{\dfrac{(1-\sin A)^2}{1-\sin^2 A}} = \sqrt{\dfrac{(1-\sin A)^2}{\cos^2 A}} = \dfrac{1-\sin A}{\cos A}$$

$$\text{RHS} = \dfrac{\cos A}{1+\sin A} = \dfrac{\cos A(1-\sin A)}{1-\sin^2 A} = \dfrac{\cos A(1-\sin A)}{\cos^2 A} = \dfrac{1-\sin A}{\cos A} \checkmark$$

</div></details>

</PYQCard>

<PYQCard year="2001" marks="4" qnum="2">

If $2\sin A - 1 = 0$, show that $\sin 3A = 3\sin A - 4\sin^3 A$.

<details><summary>Show Solution</summary><div>

$2\sin A = 1 \Rightarrow \sin A = \dfrac{1}{2} \Rightarrow A = 30°$

**LHS:** $\sin 3A = \sin 90° = 1$

**RHS:** $3\sin A - 4\sin^3 A = 3\left(\dfrac{1}{2}\right) - 4\left(\dfrac{1}{2}\right)^3 = \dfrac{3}{2} - 4 \cdot \dfrac{1}{8} = \dfrac{3}{2} - \dfrac{1}{2} = 1 \checkmark$

</div></details>

</PYQCard>

## Year 2002

<PYQCard year="2002" marks="4" qnum="1">

Prove: $\dfrac{1}{\sin A+\cos A} + \dfrac{1}{\sin A-\cos A} = \dfrac{2\sin A}{1-2\cos^2 A}$

<details><summary>Show Solution</summary><div>

$$\text{LHS} = \dfrac{(\sin A-\cos A)+(\sin A+\cos A)}{(\sin A+\cos A)(\sin A-\cos A)} = \dfrac{2\sin A}{\sin^2 A-\cos^2 A}$$

$$= \dfrac{2\sin A}{(1-\cos^2 A)-\cos^2 A} = \dfrac{2\sin A}{1-2\cos^2 A} = \text{RHS} \checkmark$$

</div></details>

</PYQCard>

<PYQCard year="2002" marks="4" qnum="2">

Prove: $\dfrac{1}{1+\sin(90°-A)} + \dfrac{1}{1-\sin(90°-A)} = 2\sec^2(90°-A)$

<details><summary>Show Solution</summary><div>

Use $\sin(90°-A) = \cos A$, $\sec(90°-A) = \operatorname{cosec} A$:

$$\text{LHS} = \dfrac{1}{1+\cos A}+\dfrac{1}{1-\cos A} = \dfrac{2}{1-\cos^2 A} = \dfrac{2}{\sin^2 A} = 2\operatorname{cosec}^2 A = 2\sec^2(90°-A) \checkmark$$

</div></details>

</PYQCard>

## Year 2003

<PYQCard year="2003" marks="4" qnum="1">

Prove: $\dfrac{\cos A}{1-\tan A} + \dfrac{\sin A}{1-\cot A} = \sin A + \cos A$

<details><summary>Show Solution</summary><div>

$$\text{LHS} = \dfrac{\cos A}{1-\frac{\sin A}{\cos A}} + \dfrac{\sin A}{1-\frac{\cos A}{\sin A}} = \dfrac{\cos^2 A}{\cos A-\sin A} + \dfrac{\sin^2 A}{\sin A-\cos A}$$

$$= \dfrac{\cos^2 A-\sin^2 A}{\cos A-\sin A} = \dfrac{(\cos A+\sin A)(\cos A-\sin A)}{\cos A-\sin A} = \cos A+\sin A = \text{RHS} \checkmark$$

</div></details>

</PYQCard>

## Year 2004

<PYQCard year="2004" marks="4" qnum="1">

Prove: $\dfrac{\cot A+\tan B}{\cot B+\tan A} = \cot A \tan B$

<details><summary>Show Solution</summary><div>

$$\text{LHS} = \dfrac{\frac{\cos A}{\sin A}+\frac{\sin B}{\cos B}}{\frac{\cos B}{\sin B}+\frac{\sin A}{\cos A}} = \dfrac{\frac{\cos A\cos B+\sin A\sin B}{\sin A\cos B}}{\frac{\cos A\cos B+\sin A\sin B}{\sin B\cos A}} = \dfrac{\sin B\cos A}{\sin A\cos B} = \dfrac{\cos A}{\sin A}\cdot\dfrac{\sin B}{\cos B} = \cot A\tan B \checkmark$$

</div></details>

</PYQCard>

<PYQCard year="2004" marks="3" qnum="2">

Evaluate: $14\sin 30° + 6\cos 60° - 5\tan 45°$

<details><summary>Show Solution</summary><div>

$$= 14\times\dfrac{1}{2} + 6\times\dfrac{1}{2} - 5\times 1 = 7 + 3 - 5 = \boxed{5}$$

</div></details>

</PYQCard>

<details><summary>Show Solution</summary><div>

$$\text{LHS} = \sec^2 A + \operatorname{cosec}^2 A = \dfrac{1}{\cos^2 A}+\dfrac{1}{\sin^2 A} = \dfrac{\sin^2 A+\cos^2 A}{\sin^2 A\cos^2 A} = \dfrac{1}{\sin^2 A\cos^2 A}$$

$$\text{RHS} = \dfrac{1}{\sin^2 A(1-\sin^2 A)} = \dfrac{1}{\sin^2 A\cos^2 A} = \text{LHS} \checkmark$$

## Year 2007

<PYQCard year="2007" marks="4" qnum="1">

Prove: $\dfrac{1}{\sec A+\tan A} = \sec A-\tan A$

<details><summary>Show Solution</summary><div>

Multiply numerator and denominator by $(\sec A - \tan A)$:

$$\text{LHS} = \dfrac{\sec A-\tan A}{\sec^2 A-\tan^2 A} = \dfrac{\sec A-\tan A}{1} = \sec A-\tan A = \text{RHS} \checkmark$$

</div></details>

</PYQCard>

<PYQCard year="2007" marks="3" qnum="2">

Evaluate: $\dfrac{\cos 70°}{\sin 20°}+\dfrac{\cos 59°}{\sin 31°}-8\sin^2 30°$

<details><summary>Show Solution</summary><div>

$\cos 70° = \sin 20°$ and $\cos 59° = \sin 31°$:

$$= \dfrac{\sin 20°}{\sin 20°}+\dfrac{\sin 31°}{\sin 31°}-8\times\left(\dfrac{1}{2}\right)^2 = 1+1-8\times\dfrac{1}{4} = 2-2 = \boxed{0}$$

</div></details>

</PYQCard>

## Year 2008

<PYQCard year="2008" marks="4" qnum="1">

Prove: $\dfrac{\sin A\tan A}{1-\cos A} = 1+\sec A$

<details><summary>Show Solution</summary><div>

$$\text{LHS} = \dfrac{\sin A\cdot\frac{\sin A}{\cos A}}{1-\cos A} = \dfrac{\sin^2 A}{\cos A(1-\cos A)} = \dfrac{1-\cos^2 A}{\cos A(1-\cos A)} = \dfrac{(1-\cos A)(1+\cos A)}{\cos A(1-\cos A)} = \dfrac{1+\cos A}{\cos A} = \dfrac{1}{\cos A}+1 = \sec A+1 \checkmark$$

</div></details>

</PYQCard>

Evaluate: $\dfrac{\cos 75°}{\sin 15°}+\dfrac{\sin 12°}{\cos 78°}-\dfrac{\cos 18°}{\sin 72°}$

$\cos 75°=\sin 15°$, $\sin 12°=\cos 78°$, $\cos 18°=\sin 72°$:

</div></details>

</div></details>

<PYQCard year="2016" marks="3" qnum="1">

Evaluate: $\operatorname{csc}^2 57°-\tan^2 33°+\cos 44°\operatorname{csc}46°-\sqrt{2}\cos 45°-\tan^2 60°$

<details><summary>Show Solution</summary><div>

- $\operatorname{csc}^2 57°-\tan^2 33°$: since $57°+33°=90°$, $\operatorname{csc}^2 57°-\cot^2 57° = 1$

</div></details>

## Year 2017

<PYQCard year="2017" marks="4" qnum="1">

Prove: $\dfrac{\sin\theta-2\sin^3\theta}{2\cos^3\theta-\cos\theta} = \tan\theta$

<details><summary>Show Solution</summary><div>

$$\text{LHS} = \dfrac{\sin\theta(1-2\sin^2\theta)}{\cos\theta(2\cos^2\theta-1)}$$

Substitute $\cos^2\theta = 1-\sin^2\theta$ in denominator: $2\cos^2\theta-1 = 1-2\sin^2\theta$

$$= \dfrac{\sin\theta(1-2\sin^2\theta)}{\cos\theta(1-2\sin^2\theta)} = \dfrac{\sin\theta}{\cos\theta} = \tan\theta = \text{RHS} \checkmark$$

</div></details>

</PYQCard>

## Year 2018

<PYQCard year="2018" marks="4" qnum="1">

Prove: $\sqrt{\sec^2\theta+\operatorname{cosec}^2\theta} = \tan\theta+\cot\theta$

<details><summary>Show Solution</summary><div>

$$\text{RHS}^2 = (\tan\theta+\cot\theta)^2 = \left(\dfrac{\sin^2\theta+\cos^2\theta}{\sin\theta\cos\theta}\right)^2 = \dfrac{1}{\sin^2\theta\cos^2\theta}$$

$$\text{LHS}^2 = \sec^2\theta+\operatorname{cosec}^2\theta = \dfrac{1}{\cos^2\theta}+\dfrac{1}{\sin^2\theta} = \dfrac{1}{\sin^2\theta\cos^2\theta}$$

Both sides positive, $\text{LHS}^2 = \text{RHS}^2$, so $\text{LHS} = \text{RHS}$ $\checkmark$

</div></details>

</PYQCard>

<PYQCard year="2018" marks="4" qnum="2">

Prove: $(1+\cot\theta-\operatorname{cosec}\theta)(1+\tan\theta+\sec\theta) = 2$

<details><summary>Show Solution</summary><div>

$$= \dfrac{\sin\theta+\cos\theta-1}{\sin\theta}\cdot\dfrac{\cos\theta+\sin\theta+1}{\cos\theta} = \dfrac{(\sin\theta+\cos\theta)^2-1}{\sin\theta\cos\theta} = \dfrac{1+2\sin\theta\cos\theta-1}{\sin\theta\cos\theta} = \dfrac{2\sin\theta\cos\theta}{\sin\theta\cos\theta} = 2 \checkmark$$

</div></details>

</PYQCard>

<details><summary>Show Solution</summary><div>

</div></details>

## Year 2024

<PYQCard year="2024" marks="4" qnum="1">

Prove: $\dfrac{\sin^3\theta+\cos^3\theta}{\sin\theta+\cos\theta}+\sin\theta\cos\theta = 1$

<details><summary>Show Solution</summary><div>

Use $a^3+b^3 = (a+b)(a^2-ab+b^2)$:

$$\text{LHS} = \dfrac{(\sin\theta+\cos\theta)(\sin^2\theta-\sin\theta\cos\theta+\cos^2\theta)}{\sin\theta+\cos\theta}+\sin\theta\cos\theta$$

$$= (1-\sin\theta\cos\theta)+\sin\theta\cos\theta = 1 = \text{RHS} \checkmark$$

</div></details>

</PYQCard>
