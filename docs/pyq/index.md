# 📋 Previous Year Questions (PYQ)

All ICSE board exam questions, organised by topic. Tap **Show Solution** only after you've attempted the question yourself.

## How to use

1. Pick a topic below
2. Read the question carefully
3. Try it on paper first
4. Tap **Show Solution** to check your working
5. Note the mark scheme — ICSE gives marks for each correct step, not just the final answer
